# Yolo NestJS Mongo POC

## Requirements

Concepts Implementation Requirements:
- API Throttling
- Rate Limiter
- Middleware
- Interceptor

Features Needed:
- User Authentication
- User Authorization
- Posts CRUD Endpoint
- Structured Response States(Both Error and Success)
- Github Actions Integration

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
